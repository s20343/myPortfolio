WHO AM I

I'm a final semester student at the Polish-Japanese Academy of Information Technology in Warsaw, Poland. I love learning new concepts and implementing them in my projects and challenges. I've developed using a diverse set of technologies, including React, Node.js, JavaScript, ASP.NET, Spring Boot, Microsoft SQL, HTML, and CSS. For my thesis , I developed a FPS/RTS (First-Person Shooter/Real-Time Strategy) 3D game using Unity.

Reach out if you'd like to learn more about my projects and see if there's anything I can do for you or view my resume below!

My Tools

See my projects below that use C# and ASP.NET Core. ASP.NET is the framework I use to build web applications in C#. Entity Frameowrk and LINQ make it easy to communicate with the database.

HTML and CSS are the basic tools needed to create a website. Bootstrap is the CSS framework I use to create responsive websites based on mobile first design.

JavaScript is used to create interactive web pages. I use JavaScript on all my projects and challenges. Take a look at my challenges to see what I can do.

PostgreSQL is the database I use on all my projects to persist data. SQL is the language that is used to communicate with the database.

Every project I work on uses git and GitHub. I love having the ability to explore new concepts in a project and then revert back to an earlier commit as needed. It also works well for CI/CD. Being able to backup and reference my code from anywhere is awesome!
